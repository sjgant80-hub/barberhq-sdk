# barberhq-sdk

**BarberHQ · The multi-location chain showroom for FallHub** — sovereign SDK extracted from `barberhq/index.html`.

## Install
```bash
npm install @ai-native-solutions/barberhq-sdk
```

## Use
```js
import * as barberhq from '@ai-native-solutions/barberhq-sdk';
```
